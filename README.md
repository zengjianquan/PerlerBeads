# 拼豆图纸生成器 (Perler Bead Pattern Generator)

一个纯前端工具，可将任意图片转换为拼豆 (Perler/Hama/MARD) 图纸，支持多品牌色号、格子级编辑与材料清单导出。

---

## ✨ 功能一览

- **图片上传与自动转换**
  - 支持点击上传或拖放图片（JPG/PNG/WebP/BMP，最大 10MB）
  - 自动将图片缩放至目标尺寸，并映射到所选品牌的拼豆颜色库

- **多种品牌颜色库**
  - **Perler** (40 色，符号 A–Z, 0–9, a–j)
  - **Hama** (40 色，符号 a–z, 0–9, A–H)
  - **MARD** (263 色，符号为色号如 A1, B3, S12 等)
  - 每个颜色都包含 RGB 值、十六进制色码、中文名称与唯一符号

- **灵活的图纸设置**
  - 调整图纸宽度与高度（8–120 珠子）
  - 限制最大使用颜色数（`自动` 或手动指定 4–40）
  - 调节预览缩放（6–35 px/珠）
  - 快速预设尺寸（29×29、50×50、80×60 等）

- **丰富的预览模式**
  - **方形像素模式**：模拟传统拼豆板
  - **圆形珠子模式**：带立体阴影与高光，更真实
  - 网格线显示/隐藏
  - 颜色符号叠加（可显示每个格子的符号）
  - 图例：展示所有用到的颜色及其符号与用量
  - 行列号标注：边缘显示行号和列号，便于定位

- **材料清单**
  - 自动统计每种颜色的用量，按使用频率排序
  - 显示总计珠子数量和颜色种数
  - 每个颜色展示色块、名称、符号和数量

- **图纸导出**
  - 导出为高分辨率 PNG（30 px/珠）
  - 可选择导出“彩色图纸”或“符号图纸”
  - 导出时保留当前的行列号、图例等设置
  - 快捷键：`Ctrl+E` 导出彩色，`Ctrl+S` 导出符号

- **格子级手动编辑**
  - 点击图纸上的任意格子即可选中（红色虚线高亮）
  - 左侧编辑面板显示当前格子的坐标与颜色
  - 通过品牌色板直接点击色块替换
  - 使用系统取色器选取颜色，**自动匹配**到当前品牌最接近的色号

- **打印友好**
  - 打印时自动隐藏侧边栏和控制按钮
  - 只保留纯黑或纯白高对比度符号

- **演示图案**
  - 启动后若无上传图片，5 秒后自动加载一个彩色花朵图案，方便快速体验

## 🛠️ 技术栈

- **纯前端**：HTML5 + CSS3 + JavaScript (ES6)
- **Canvas 2D API**：图像处理与图纸渲染
- **Flexbox 布局**：响应式界面，适配桌面与平板
- **CSS 变量**：统一维护主题色与间距
- **无外部依赖**：完全自包含单文件，可直接在浏览器中运行

## 🧠 核心算法

### 1. 图像自适应裁剪 (Cover Mode)
保持原图宽高比，居中裁剪出与目标图纸尺寸比例匹配的区域，避免图像拉伸变形。

### 2. 颜色匹配（加权 RGB 距离）
为每个像素找到最接近的拼豆颜色，采用加权欧几里得距离公式：

$$
distance = 0.299 \times (R1-R2)^2 + 0.587 \times (G1-G2)^2 + 0.114 \times (B1-B2)^2
$$

权重基于人眼对红、绿、蓝不同敏感度（标准 luminance 系数），提高匹配的自然度。

### 3. 颜色数量限制
当实际使用颜色超过设定上限时：
- 按颜色使用频率从高到低排序，保留前 N 种颜色
- 其余像素重新映射到保留颜色中欧氏距离最近的颜色

### 4. 取色器智能匹配
用户在编辑格子时使用 `<input type="color">` 选取任意颜色，工具立即计算该颜色到当前品牌所有色号的距离，并替换为最近的品牌色号，保证最终图纸的可制作性。

### 5. 高分辨率导出
导出时将预览缩放临时提升至 30 px/珠，生成高清图纸，然后恢复原始缩放状态，不影响当前视图。

## 📖 使用方法

1. 打开 `index.html`（或直接在浏览器中运行该 HTML 文件）
2. 在左侧面板**上传一张图片**，或等待演示图案自动加载
3. 选择**品牌**（Perler / Hama / MARD）
4. 调整**图纸尺寸**、**最大颜色数**、**显示缩放**等参数
5. 根据需要开启/关闭网格线、圆形模式、符号叠加、行列号等
6. （可选）点击右侧图纸上的某个格子，然后在左侧色板或取色器中修改该格子的颜色
7. 点击**导出彩色图纸**或**导出符号图纸**按钮保存 PNG
8. 材料清单会自动更新，可对照购买相应色号的拼豆

## ⚙️ 开发说明

- **颜色库维护**：颜色数据定义在 `BASE_COLORS`、`BRAND_SYMBOLS` 和 `MARD_COLORS` 变量中，可按需增删改
- **导出分辨率**：修改 `exportPNG` 函数内的 `exportZoom` 变量（默认 30）可调整导出清晰度
- **颜色距离权重**：在 `rgbDistance` 函数中可微调系数以优化匹配效果
- **最大尺寸限制**：为防止性能问题，图纸最大宽度和高度设定为 120，可在 `<input>` 的 `max` 属性中修改
